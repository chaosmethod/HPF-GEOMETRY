# Scalar sector

$∇·E_g = -4πG ρ_HPF$

$E_g = -∇Φ_g$

---

# Vector (gravitomagnetic) sector

$B_g = ∇×A_g$

$∇·B_g = 0$

$∇×B_g = (16πG / c^2) j_HPF$

---

# Time-dependent extension

$∇×E_g = -∂_t B_g$

$∇×B_g =
-(4 / c^2) ∂_t E_g
+ (16πG / c^2) j_HPF$

---

# Interpretation

$ρ_HPF$ = effective load density 

$j_HPF$ = directional load transport
