# Spherical source

$m_HPF(r) =
4π ∫₀^r ρ_HPF(r') r'^2 dr'$

$Λ_r(r) = 2G m_HPF(r) / (r c^2)$

---

# Metric

$g_{00} = -(1 - Λ_0)^2$
$g_{rr} = 1 / (1 - Λ_r)$

---

# Constant density

$ρ = ρ₀$

$Λ_r = (8πGρ₀ / 3c^2) r^2$

---

# Rotation (Kerr-like)

$Λ_{0φ} ≠ 0$

$g_{tφ} sourced by Λ_{0φ}$

Outside solution:

$g_{tφ} = - (2GJ / c^3 r) sin²θ$
