# Substrate Load Definition

Microscopic variable:

$Λ_{μν}^{(sub)}(x)$

Properties:
- dimensionless
- bounded: $0 ≤ Λ ≤ 1$
- local
- conserved:

$∇_μ Λ^{(sub)μν} = 0$

---

Interpretation:

$Λ_{μν}$ encodes directional depletion of update capacity.

$Λ_00 = Λ_0$ → temporal capacity depletion

$Λ_0i$ → directional load flow

$Λ_ij$ → anisotropic transport load
