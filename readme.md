Canonical consolidation for the **Holographic Projection Framework (HPF)**, the **Mixture-of-Domain-Experts Architecture (MDEA)** routing kernel, the **geometry/gravity effective expert**, and the **geometry-failure / percolation transition model**.

This repository is not a single object-level gravity model. It is a layered framework package:

- **HPF** is the regulatory legality framework / physics OS.
- **MDEA** is the routing kernel that selects the active expert.
- **Geometry / gravity** is an effective expert regime valid only in its domain.
- **QPRCA** is the deeper substrate-level handoff destination when geometry can no longer be trusted.

That separation is canonical and should not be collapsed.

HPF starts from bounded physical execution rather than unconstrained continuum ontology. The core axioms are:

- finite capacity
- finite resolution
- locality
- reversibility
- conservation in the effective load description

The primitive substrate object is the local bounded load tensor:


 $\Lambda^{(\mathrm{sub})}_{\mu\nu}(x)$


with the canonical temporal health variable:


$G_{\mathrm{health}} = 1 - \Lambda_0$


The central framework distinction is:

- **Legality** = whether substrate evolution is admissible under HPF constraints
- **Validity** = whether a chosen expert remains trustworthy in its own regime

These are not the same thing.

- baseline smooth-law spanning threshold: **~0.50**
